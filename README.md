# maeManagerBackend

### Tecnologias 

  - Node 
  - Express 
  - MongoDB con Mongoose 
 
### Deploy con Heroku 

URL : https://maemanager.herokuapp.com

### Endpoints disponibles 
  - /api/tasks
  - /api/fondos
  
### Clonar el Repo, luego instalar y correr el proyecto
  - git clone 
  - npm install 
  - npm start
